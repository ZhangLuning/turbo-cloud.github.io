简介

Turbo Cloud 是一个基于前端：Vue3.0、Ant-Design-Vue、TypeScript；后台：SpringCloud、SpringBoot、MyBatis-Plus 前后端分离的微服务框架。

愿景

我们的愿景是成为 Spring Cloud 最好的增强，帮助公司快速的搭建项目框架，效率翻倍。

# 项目概览

  • Turbo Cloud (opens new window)：基于 SpringCloud、SpringBoot、MyBatis-Plus 微服务后台.
  • Turbo Cloud Gateway (opens new window)：使用Netty高性能的通信框架，Webflux响应式编程，独立出来网关。
  • Turbo Cloud Front Vue (opens new window)：基于 Vue3.0、Vite、 Ant-Design-Vue、TypeScript 微服务前端。
  • Turbo Cloud Blog Doc (opens new window)：Turbo Cloud微服务的完整文档。

# 数据库

使用 MyBatis 进行 CRUD, 并且支持标准 SQL 的数据库。

  • 项目中目前使用了MySQL数据库
